# Инструкция по развертыванию проекта (Setup)

## Ключи для базы данных и Yandex Cloud Object Storage расположены в .env

## Перед запуском сервера установите все необходимые пакеты:

```bash
npm install
```

## Примените схему данных Drizzle к вашей локальной базе данных SQLite:

```bash
npm run db:push
```

## Запустите локальный сервер Nuxt 4 / Nitro:

```bash
npm run dev
```

После запуска проект будет доступен по адресу: [http://localhost:3000]

# HintBook — fullstack-приложение, построенное на базе экосистемы Vue.

### Фронтенд (Frontend)
* **Nuxt 4 / Vue 3** — фреймворк для создания быстрого пользовательского интерфейса. Поддерживает серверный рендеринг (SSR).
* **Спецификация App Directory** — вся фронтенд-логика изолирована внутри директории `app/`.

### Бэкенд (Backend) & База данных
* **Nitro** — встроенный в Nuxt высокопроизводительный веб-сервер. Обрабатывает API-запросы (`server/api/`).
* **PostgreSQL** — объектно-реляционная база данных. Хранит информацию о пользователях и работах.
* **Drizzle ORM** — TypeScript-first инструмент для работы с реляционными компонентами баз данных. Описывает схемы (`schema.ts`) и выполняет миграци.

### Облачные сервисы (Инфраструктура)
* **Yandex Cloud Object Storage** — S3-совместимое облачное хранилище. Используется для хранения и раздачи загружаемых файлов.



```text
hintbook/
├── .env                        # Секретные ключи (БД, Yandex Cloud)
├── drizzle.config.ts           # Конфигурация базы данных Drizzle
├── nuxt.config.ts              # Главный конфиг Nuxt 4
├── package.json                # Зависимости проекта
├── app/                        # --- ФРОНТЕНД (VUE 3) ---
│   ├── app.vue                 # Главный компонент-оболочка
│   ├── components/
│   │   └── AppHeader.vue       # Шапка сайта
│   └── pages/
│       ├── index.vue           # Главная страница (Список работ)
│       └── loadwork.vue        # Страница загрузки работы
└── server/                     # --- БЭКЕНД (NITRO) и БАЗА ДАННЫХ (POSTGRESQL)---
    ├── api/
    │   ├── works.get.ts        # Получение списка работ из БД
    │   └── upload.post.ts      # Загрузка файла в Yandex и запись в БД
    ├── database/
    │   ├── db.ts               # Инициализация подключения к PostgreSQL
    │   └── schema.ts           # Схема таблиц БД (Drizzle)
    └── utils/
        └── s3.ts               # Утилита для работы с Yandex Object Storage
```
